Personal AI chats where clear text and readable answers always come first.

Features

3 Built-in Backgrounds

Upload Your Own

Use an image from your computer as a chat background. Large files are optimized locally for reliable storage and faster loading.

Solid Color Backgrounds

Switch to clean solid surfaces when artwork feels too busy. Great for paper-like reading modes and distraction-free contrast.

✨

Smooth Animations

Backgrounds fade in over 2 seconds. Switching themes plays a fade-out / fade-in transition, while reduced-motion preferences are respected automatically.

💬

Custom Bubble Colors

Pick separate colors for your messages and the assistant from quick palettes or direct color inputs.

👁

Always-on Readability Guard

Every style is checked against a 4.5:1 contrast target across chat text, generated answers, sidebars, inputs, and document windows.

📄

Themed Document Windows

Canvas, Artifacts, markdown files, dialogs, and file previews keep your chosen colors, opacity, and readable text sizing.

🔒

Privacy First

No data leaves your browser. No analytics, no tracking, no server calls. Fully open source under MIT.

Accessibility Studio

Customize Freely. Readability Stays Protected.

Version 1.3.8 evaluates each translucent surface before it reaches your chat. It keeps your preferred color whenever it is safe, raises only the opacity that needs protection, and changes text to black or white only as a last resort.

Generated answers remain clear

Assistant responses, nested markdown, headings, lists, links, and code blocks inherit a protected readable color instead of a site default.

Protection is surface by surface

Your messages, AI output, composer, sidebar, Canvas, Artifacts, markdown files, and previews are evaluated independently.

No hidden compromise

The popup preview reports when Readability Guard protects opacity or corrects an unsafe text color before you click Apply Style.

Install in 30 Seconds

  1. 1
    Download

    Clone or download the repo from GitHub.

  2. 2
    Open Extensions

    Go to chrome://extensions and enable Developer mode.

  3. 3
    Load Unpacked

    Click Load unpacked and select the project folder.

  4. 4
    Style Your Chat

    Click the extension icon, choose a background if you want one, tap a preset for an instant reading mode, or adjust text size, colors, opacity, and brightness manually before clicking Apply Style. The popup confirms what happened.
