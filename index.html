My Chat Faces

Accessible backgrounds, solid colors, and readable chat styling for ChatGPT, Claude, Gemini & Grok.

Start with a built-in theme, a solid paper-like surface, or your own image, then tune bubble colors, text size, contrast, opacity, and background brightness so every chat stays readable.

Install in 30 Seconds

  1. 1
    Download

    Clone or download the repo from GitHub.

  2. 2
    Open Extensions

    Go to chrome://extensions and enable Developer mode.

  3. 3
    Load Unpacked

    Click Load unpacked and select the project folder.

  4. 4
    Style Your Chat

    Click the extension icon, choose a background if you want one, tap a preset for an instant reading mode, or adjust text size, colors, opacity, and brightness manually before clicking Apply Style.
